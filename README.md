# 📊 Student Performance Analysis

## 📌 Project Overview

This project analyzes student performance using Python. The dataset includes student details like study time, grades, and demographics.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib

## 📂 Dataset

* student-mat.csv

## 📊 Key Insights

* Average performance is around 10.4 marks
* Only a small number of students scored above 15
* Study time has a weak impact on performance
* Male students perform slightly better than female students

## 📈 Visualizations

* Histogram of grades
* Scatter plot (study time vs grades)
* Bar chart (gender vs performance)

## 🚀 Conclusion

Student performance is moderate and influenced by multiple factors beyond study time.
